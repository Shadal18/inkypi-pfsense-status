API key setup
This plugin requires a pfSense REST API key.
Create the key in pfSense under System > REST API > Keys.
To add the key in InkyPi, open the front page, click the key icon, add a new key named PFSENSE_API_KEY, paste in your pfSense API key, and save it.
Restart InkyPi if needed after saving the key.
📝
Header
Set the title shown at the top of the status panel.

This text appears as the main heading on the rendered status panel.

📊
Display cards
Choose which pfSense status cards appear on the display.
Clients
Show connected or active client information.
Uptime
Show pfSense uptime on the dashboard.
CPU
Show current CPU usage information.
Memory
Show current memory usage information.
Temperature
Show temperature data when available.
Interfaces list
Show the network interfaces summary or list.

Uncheck any cards you do not want to show on the display.
